Lost and Found App

Overview

This is an Android application developed using Java and SQLite that allows users to create, view, filter, and manage lost and found item 
adverts. The app is designed to help reconnect lost items with their owners in a simple and user-friendly way.


Create a new advert (name, description, category, image)

Store data using SQLite database
View all lost and found items
Filter items by category
Timestamp for each advert
Image upload and preview
Detailed view of each item
Remove advert once item is resolved


Technologies Used

Java (Android Development)
SQLite (Local Database)
Android Studio
XML (UI Design)


How to Run

Open the project in Android Studio
Connect an emulator or Android device
Click Run 
Start using the app

How Image Upload Works

User selects image from device storage
Image is stored as URI (not actual file)
URI is saved in SQLite
Image is displayed using ImageView
